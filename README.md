## Description
An interactive Computer Graphics application using Python and PyOpenGL.
The app visualizes the impact of energy consumption through a toggleable lightbulb and a dynamic energy meter

## How to Run
1. Install requirements: `pip install PyOpenGL`
2. Run the script: `python EnergySavingAssignment.py`
3. Controls: Press **'S'** or **'L'** to toggle energy saving mode
