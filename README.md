## 📝 Project Description

This is an interactive Computer Graphics application designed to raise awareness about electricity consumption. Built using **Python** and **PyOpenGL (GLUT)**, the app visualizes:

- **High Consumption:** A bright room with a red energy meter
- **Energy Saving:** A dimmed room with a green "eco-friendly" meter


The project fulfills the university's creative requirement to encourage wise electricity usage under the "Wafarha w Nawarha" campaign

## 🕹️ Controls

- **Press 'S' or 'L'**: Toggle Energy Saving Mode (Light On/Off)
- **Press 'Space'**: Alternative toggle


## 🛠️ Technical Details (For Developers)

If you wish to run the source code directly:

1. Ensure Python 3.14+ is installed
2. Install dependencies:
    ```bash
    pip install PyOpenGL PyOpenGL_accelerate
    ```
3. Keep `freeglut.dll` in the same directory as the script
4. Run the script:
    ```bash
    python EnergySavingAssignment.py
    ```

## 📂 Project Structure

- `EnergySavingAssignment.py`: Main source code
- `freeglut.dll`: OpenGL utility toolkit for Windows
